# Customer Segmentation Dashboard

This is a professional-grade Streamlit application designed for shopkeepers and businesses to analyze customer data using K-Means clustering.

## 🚀 Features
* **Automated Data Cleaning:** Handles missing values and outliers.
* **Smart Feature Engineering:** Auto-selects relevant columns.
* **Optimal Clustering:** Uses Elbow Method and Silhouette Score.
* **Interactive Visualizations:** 2D/3D PCA plots and cluster profiling.
* **Business Recommendations:** Generates actionable strategies.

## 🛠️ Installation
1. Ensure Python 3.9+ is installed.
2. Install dependencies: `pip install -r requirements.txt`

## ▶️ Usage
1. Run the app: `streamlit run app.py`
2. Upload data (CSV/Excel) in the **Upload** tab.
3. Configure and run analysis in the **Analysis** tab.
4. View results and download segments.
